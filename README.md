## Full Stack Developer Portfolio Website
 Engineered a modern full-stack portfolio web application with a responsive and interactive user interface using React.js and Tailwind CSS. Designed and developed RESTful backend APIs using Node.js and Express.js to manage portfolio contact form submissions efficiently. Integrated MongoDB Atlas for real-time data storage and retrieval.

## Overview

A modern and responsive full-stack portfolio web application developed using React.js, Node.js, Express.js, MongoDB Atlas, Mongoose, and Tailwind CSS. This project was designed to showcase technical skills, projects, certifications, and professional information through an interactive and production-ready developer portfolio platform.

The application includes a complete frontend and backend architecture with dynamic contact form functionality, cloud database integration, REST API communication, and responsive user interface design.

## Project Architecture

Frontend developed using React.js communicates with backend REST APIs built using Express.js and Node.js. Backend services are connected to MongoDB Atlas cloud database using Mongoose ODM for secure and scalable data storage.

---

## Contact Form Workflow

1. User submits form from frontend.
2. React frontend sends API request.
3. Express backend receives form data.
4. MongoDB Atlas stores submitted data.
5. Backend returns success response.

## Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Itzshambhav/portfolio-website.git
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm start
```

---

## Environment Variables

Create `.env` file inside `server` folder:

```env
PORT=5000
atlas_URL=YOUR_MONGODB_CONNECTION_STRING
```
